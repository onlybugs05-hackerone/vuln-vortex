# Security Policy

## Supported Versions

The latest version of VULN-VORTEX is always supported. Please ensure you are running the latest commit from the `main` branch.

## Reporting a Vulnerability

If you discover a security vulnerability within VULN-VORTEX itself (e.g., unintended code execution, sensitive data leak), please report it via:

1.  **GitHub Issues**: Open an issue labeled "Security".
2.  **Email**: Contact OnlyBugs05 directly (if public).

We appreciate your responsible disclosure and will address the issue promptly.

## Safe Usage Disclaimer

This tool is designed for **authorized security testing only**. The author is not responsible for any misuse or damage caused by this tool. Always obtain permission before scanning a target.
